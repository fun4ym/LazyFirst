## 产品概述

统一系统中所有展示商品信息的列表样式为截图所示样式，采用左侧商品图片 + 右侧信息（紫色商品ID、商品名称、带店铺图标的绿色店铺名称）的布局。

## 核心功能

- 修改5个页面的商品信息列显示样式
- 统一样式：左侧el-image商品图片，右侧垂直排列商品ID（紫色#9C27B0）、商品名称（灰色#595959）、店铺名称（带Shop图标，绿色#6DAD19）
- 保持原有功能：tooltip提示、点击跳转、图片预览等

## 技术栈

- 前端框架：Vue 3 + Element Plus
- 样式语言：CSS/SCSS
- 已有基础样式：`/frontend/src/styles/common.css` 中的 `.product-info` 等

## 实现方案

### 统一样式结构

```
<div class="product-cell">
  <el-image :src="图片字段" fit="cover" class="product-thumb" />
  <div class="product-info">
    <div class="product-id purple">{{ 商品ID字段 || '--' }}</div>
    <el-tooltip :content="商品名称字段" placement="top">
      <div class="product-name">{{ truncateText(商品名称字段, 50) }}</div>
    </el-tooltip>
    <div class="shop-name">
      <el-icon><Shop /></el-icon>
      {{ 店铺名称字段 || '--' }}
    </div>
  </div>
</div>
```

### 字段映射

| 页面 | 图片字段 | 商品ID字段 | 商品名称字段 | 店铺名称字段 |
| --- | --- | --- | --- | --- |
| ProductTab.vue | row.images | row.tiktokProductId | row.name | row.shopId?.shopName |
| Management.vue | row.productImage | row.productId_display | row.productName | row.shopName |
| ManagementBDSelf.vue | row.productImage | row.productId_display | row.productName | row.shopName |
| report-orders/Index.vue | row.productImage | row.productId | row.productName | row.shopName |
| PublicCollection.vue | row.productImage | row.productId | row.productName | row.shopName |


### CSS样式新增

在 `common.css` 或各文件 `<style>` 中添加：

```css
.product-cell {
  display: flex;
  align-items: flex-start;
  gap: 12px;
}
.product-thumb {
  width: 60px;
  height: 60px;
  border-radius: 4px;
  border: 1px solid #ebeef5;
}
.product-id.purple {
  color: #9C27B0;
  font-size: 13px;
  font-weight: 500;
}
.shop-name {
  display: flex;
  align-items: center;
  gap: 4px;
  font-size: 12px;
  color: #6DAD19;
}
```

## 设计风格

采用截图中的简洁列表展示风格：

- 左侧商品缩略图（60x60px圆角矩形）
- 右侧信息垂直排列：
- TikTok商品ID（紫色#9C27B0，13px）
- 商品名称（灰色#595959，12px，带tooltip截断）
- 店铺名称（绿色#6DAD19，12px，带Shop图标）
