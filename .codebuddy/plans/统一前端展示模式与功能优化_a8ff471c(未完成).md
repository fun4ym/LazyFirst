## 产品概述

统一前端展示模式与功能优化计划，旨在提升系统组件复用性、视觉一致性和用户体验。

## 核心功能

### 1. 统一前端展示组件

- **商品信息展示组件**：统一/samples "商品信息"列、/products "TikTok商品"列、/orders "商品"列的展示样式
- **达人信息展示组件**：统一/orders "TikTok ID"列、/samples "TikTok ID"列、/samples/public "达人信息"列的展示样式
- **组件特性**：
- TikTokID显示 + 一键复制icon
- 第二行显示GMV、粉丝数、月均播、月销量
- 统一视觉样式和交互体验

### 2. /samples/public页面增强

- 增加"商家视角"页签，展示产品相关的视频信息、投流码、达人信息
- 方便商家操作和查看产品推广情况

### 3. BD匹配逻辑优化

- 将/orders的BD匹配从"通过申样匹配BD"改为"订单和视频匹配出BD"
- 提高BD匹配的准确性和实用性

### 4. /videos列表完善

- 添加influencer列，显示达人信息
- 完善视频登记信息的完整性

## 技术栈

- **前端框架**：Vue 3 + Composition API
- **UI组件库**：Element Plus
- **构建工具**：Vite
- **后端框架**：Node.js + Express
- **数据库**：MongoDB + Mongoose

## 实现方案

### 1. 组件化设计

#### ProductCell组件

- **props**: `product` (对象，包含id, name, tiktokProductId, images, shopName等)
- **功能**: 商品图片预览、TikTokID显示与复制、商品名称、店铺信息
- **样式**: 统一商品信息展示样式，复用现有product-cell样式

#### InfluencerCell组件  

- **props**: `influencer` (对象，包含tiktokId, tiktokName, latestFollowers, latestGmv, monthlySalesCount, avgVideoViews等)
- **功能**: TikTokID显示与复制、达人昵称、GMV/粉丝数/月均播/月销量统计
- **样式**: 统一达人信息展示样式
