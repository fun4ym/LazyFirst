## 页面路径
- 订单管理页面: `http://localhost:5173/report-orders`

## 文件名示例
```

```

## 导入规则

1. **文件格式**: 仅支持 `.xlsx` 和 `.xls` 格式
2. **字段匹配**: 根据字段名(第一行)自动匹配数据
3. **数据处理**:
   - 订单ID重复时会更新已存在的订单
   - 空值字段会被忽略
   - 金额字段自动转换为数字
   - 日期字段自动识别并转换为日期格式
4. **字符编码**: 请使用UTF-8编码

## 注意事项

- 订单ID是必填字段，作为唯一标识
- 佣金率字段请使用小数格式(如 0.15 表示15%)
- 日期格式建议使用 Excel 支持的标准日期格式
- 导入过程中如果某行数据格式错误，该行会被跳过并记录错误信息

## 前端列表展示说明

### 列表列说明

1. **达人**: 显示达人用户名
2. **订单**: 
   - 第一行: 订单ID (字号10)
   - 第二行: 子订单ID (字号8)
3. **商品**:
   - 第一行: 商品ID + SKU (字号10)
   - 第二行: 商品名称 (字号8, 超过16字符截断为...)
   - 鼠标移入显示完整商品名称和店铺代码
4. **交易**:
   - 商品价格和下单件数
   - 订单状态用icon表示(进行中/已完成)
5. **佣金**: 联盟合作伙伴实际佣金 + 实际联盟服务商店铺广告佣金
6. **佣金率**:
   - 联: 联盟合作伙伴佣金率
   - 创: 创作者佣金率
   - 服: 服务商奖励佣金率
   - 达: 达人奖励佣金率
7. **广告佣金率**:
   - 联服: 联盟服务商店铺广告佣金率
   - 达: 达人店铺广告佣金率
8. **时间**:
   - 创: 创建时间
   - 达: 订单送达时间
   - 无数据显示 "--"
9. **打款**:
   - 第一行: 佣金结算时间
   - 第二行: 打款单号
