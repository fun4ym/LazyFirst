## 二、后台测试步骤（请主人登录后操作）

登录地址：https://tap.lazyfirst.com （用管理员账号）

### 1️⃣ 产品推送（对应"新品上架 / 活动商品"紫色卡片）

1. 左侧菜单 → **产品（Products）** → 产品列表（ProductTab）
2. 在任意一行的最右「操作」列，点 **「推送」** 按钮（即 `linePush.productPushTitle`）
3. 弹出推送对话框，左侧为受众筛选：
   - **品类标签**（categoryTags）：可多选，留空=不限
   - **适用品类**（suitableCategories）：可多选，留空=不限
   - **粉丝区间**（followerMin ~ followerMax）：留空=不限
4. 点 **「预览人数」** 按钮，查看匹配到的达人数量（显示为 `匹配：N 人`）
5. 右侧实时预览卡片样式（🆕 NEW ARRIVAL 紫头 + 商品主图 + 价格 + 佣金）
6. 点 **「确认推送」**（若匹配人数为 0 则按钮置灰不可点）
7. 卡片即推送到符合受众条件的达人 LINE

### 2️⃣ 活动推送（对应"新活动来袭"深紫色卡片）

1. 左侧菜单 → **活动（Activities）** → 活动列表
2. 在任意一行「操作」列，点 **「推送 LINE」** 按钮（即 `linePush.btn`）
3. 弹出推送对话框：
   - 左侧：受众筛选（品类标签 / 适用品类 / 粉丝区间）
   - 左侧下拉：**选择该活动下的商品**（右侧预览对应商品卡）
   - 推送模式：**Multicast**（默认）/ **Narrowcast**（匹配人数 ≥50 才可切）
4. 点 **「预览人数」** 看匹配数
5. 点 **「确认推送」** → 弹二次确认框 → 确认
6. 活动卡片推送到受众达人 LINE

### 3️⃣ 发送记录查看

1. 在 **活动（Activities）列表页右上角**，点 **「发送记录」** 按钮（即 `lineRecords.open`）
2. 弹出记录弹层，有三个页签：
   - **全部** / **活动**（campaign）/ **商品**（product）
3. 每条记录显示：时间、类型、目标对象、操作人、受众条件摘要、接收人数、状态
   - 状态：✅ success / ⚠️ partial（部分失败）/ ❌ failed
4. 超过 20 条有分页

### 4️⃣ 申样通知（自动触发，仅推卖家 —— 本次改造重点）

- **没有手动按钮**，由达人提交申样时自动触发
- 测试路径：用店铺识别码打开公开申样页 `https://tap.lazyfirst.com/samples/public?s=XXX`（XXX 为店铺识别码，可在产品/店铺页复制申样链接）
- 达人在公开页提交申样 → 系统自动推送 **「新的申样申请」橙色卡片** 给**卖家**（shop contact），**不再推送给达人**
- 可在卖家的 LINE 上验收这张橙色卡片
