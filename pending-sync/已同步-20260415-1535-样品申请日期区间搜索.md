# 样品申请日期区间搜索功能

## 修改时间
2026-04-15 15:35

## 修改内容
将样品申请列表的"申请日期"搜索从单个日期选择改为日期区间选择

## 修改文件

### 前端文件
1. `frontend/src/views/samples/Management.vue`
   - 修改日期选择器为日期区间选择器 (`type="daterange"`)
   - 修改 `searchForm` 数据结构：`date: ''` → `dateRange: []`
   - 修改 `loadSamples()` 函数：将 `dateRange` 转换为 `dateStart` 和 `dateEnd` 参数
   - 修改 `resetSearch()` 函数：重置 `dateRange` 为空数组
   - 日期区间选择器宽度设为 240px

2. `frontend/src/i18n/zh.js`
   - 添加 `startDate: '开始日期'`
   - 添加 `endDate: '结束日期'`

3. `frontend/src/i18n/en.js`
   - 添加 `startDate: 'Start Date'`
   - 添加 `endDate: 'End Date'`

### 后端文件
1. `server/routes/samples.js`
   - 添加 `dateStart` 和 `dateEnd` 参数支持
   - 实现日期区间查询逻辑：`dateStart` 到 `dateEnd` 的范围查询
   - 保留单个 `date` 参数查询的向后兼容性

## 功能说明
- 用户可以选择日期区间进行搜索
- 支持清除选择
- 日期格式为 YYYY-MM-DD
- 日期区间查询为闭区间（包含开始和结束日期）
