## 修改内容

### 样品数据清理脚本

**功能**：
1. 删除无法按名称匹配的孤立记录（productId 是24位十六进制ObjectId但商品表中找不到对应商品）
2. 根据 productId + date + influencerAccount 去重，保留 sampleStatus 权重最高的记录
   - 权重: pending(0) < sent(1) < received(2) = refused(2)

**使用方式**：
```bash
cd server && node cleanup_sample_data.js
```

**注意**：
- 这是数据清理脚本，需要在服务器上运行
- 运行前建议先备份数据库
- 操作对象是 MongoDB tap_system 数据库
