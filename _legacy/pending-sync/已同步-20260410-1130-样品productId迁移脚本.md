## 修改内容

### 通过商品名称匹配迁移孤立样品记录

**功能**：
1. 查找孤立样品记录（productId 是24位十六进制ObjectId）
2. 根据 productName 在商品表中查找对应的 tiktokProductId
3. 如果找到匹配，更新 productId；如果存在重复则删除当前记录

**使用方式**：
```bash
cd server && node migrate_sample_by_name.js
```

**注意**：
- 这是数据迁移脚本，需要在服务器上运行
- 运行前建议先备份数据库
- 与 `cleanup_sample_data.js` 配合使用
- 操作对象是 MongoDB tap_system 数据库
