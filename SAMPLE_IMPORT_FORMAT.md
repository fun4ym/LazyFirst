| 字段名称 | 是否必填 | 字段说明 | 示例 |
|---------|---------|---------|------|
| 日期 | 是 | 寄样日期 | 01/03/2026 |

| 商品ID | 是 | 商品ID | 1732233191748699642 |
| 达人账号 | 是 | 达人TikTok账号 | @username |
| 粉丝数 | 否 | 达人粉丝数 | 10000 |
| 归属业务员 | 否 | 业务员姓名 | 张三 |

| 是否出单 | 否 | 是否产生订单（是/否） | 是 |

## 数据规则

1. **唯一键**：`日期` + `达人账号` + `商品ID` 组合必须唯一
2. **重复记录**：如果导入的数据唯一键已存在，则该行导入失败
3. **日期格式**：支持 `DD/MM/YYYY` 或 `DD/MM/YYYY HH:mm:ss` 格式
4. **是否类字段**：支持 "是"/"否"、true/false、1/0

## 导入结果说明

导入完成后，系统会返回以下信息：
- **新增**：成功导入的新记录数量
- **更新**：更新的记录数量（当前为0，重复记录直接失败）
- **失败**：失败的记录数量
- **错误详情**：每条失败记录的行号和具体错误原因

## 注意事项

1. 导入前请确保Excel文件第一行为表头
2. 日期列请使用标准的 DD/MM/YYYY 格式
3. "是否"类字段请使用"是"或"否"
4. 重复的记录不会覆盖，直接标记为失败
